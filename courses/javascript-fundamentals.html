What is JavaScript?

JavaScript is a programming language that brings interactivity to web pages. It runs in browsers and enables dynamic content.

Variables & Data Types

// Variable declaration
const name = 'Fouad';  // String
let age = 28;          // Number
var active = true;     // Boolean

// Arrays
const skills = ['Python', 'JavaScript', 'React'];

// Objects
const person = {
    name: 'Fouad',
    age: 28,
    skills: ['Python', 'JavaScript']
};

Functions & Callbacks

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow function
const multiply = (a, b) => a * b;

// Callback
array.map(item => item * 2);

Async & Promises

async/await and Promises help handle asynchronous operations like API calls and file reading.
