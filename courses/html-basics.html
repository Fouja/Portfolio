What is HTML?

HTML (HyperText Markup Language) is the foundation of all websites. It provides the structure and content of web pages.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Common HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <ul>, <ol>, <li> - Lists
  • <div> - Containers

Semantic HTML

Use semantic tags like <header>, <nav>, <main>, <footer> for better structure and accessibility.
