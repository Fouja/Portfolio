What is CSS?

CSS (Cascading Style Sheets) controls the presentation and layout of HTML elements. It allows you to create beautiful, responsive designs.

CSS Selectors

/* Element Selector */
p { color: blue; }

/* Class Selector */
.highlight { background: yellow; }

/* ID Selector */
#header { font-size: 24px; }

/* Attribute Selector */
input[type="text"] { border: 1px solid gray; }

Box Model

Every HTML element is a box with margin, border, padding, and content. Understanding the box model is crucial for layout.

.box {
    margin: 20px;
    border: 2px solid black;
    padding: 15px;
    width: 200px;
}

Flexbox & Grid

display: flex and display: grid are powerful tools for creating responsive layouts without floats.
